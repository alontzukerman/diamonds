# Diamonds are Forever

A diamond ring and proposal configurator web application.

---

## Project Overview

This project allows users to design their perfect proposal by configuring:
- The engagement ring (material)
- The diamond
- The proposal location
- The setup/ambiance
- Background music
- Timeline/deadline

As users make selections, a "Love Meter" gauge fills up to show "how much he loves me."

## Quick Start

1. Open `index.html` in a browser to view the landing page
2. Click "Start" to navigate to the configurator
3. Work through each step to design your proposal

---

## Technical Stack

- Pure HTML/CSS/JS (no frameworks)
- CSS custom properties for theming
- Flexbox for layouts
- CSS animations for transitions
- Custom cursor implemented via JavaScript
